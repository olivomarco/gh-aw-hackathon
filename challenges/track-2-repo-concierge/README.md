# Track 2 — Repo Concierge 🟡

*Event-driven automation: issue triage, PR review, slash commands.*

## Challenges

| # | Challenge | Concepts |
|---|-----------|----------|
| [2-01](challenges/2-01/) | Triage Bot | Issue classification, priority labels, AI reasoning |
| [2-02](challenges/2-02/) | Review Buddy | PR events, inline comments, diff analysis |
| [2-03](challenges/2-03/) | Slash & Burn | Slash commands, comment triggers, deployment |
| [2-04](challenges/2-04/) | Stale Patrol | Scheduled + conditional logic, issue closing |
| [2-05](challenges/2-05/) | Welcome Wagon | First-time contributor detection, personalized responses |

## Prerequisites

- Completed at least 2 challenges from Track 1
- Comfortable with workflow Markdown syntax and safe-outputs

## What You'll Learn

- Event-driven workflows (issues, PRs, comments)
- Slash command patterns (comment-triggered automation)
- AI-powered decision making in workflows
- Composing multiple steps with conditional logic

## Time Budget

~120 minutes. Most squads complete 3 of 5 challenges.
