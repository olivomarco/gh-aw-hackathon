# Track 1 — Hello, Agent 🟢

*First steps with agentic workflows: schedules, safe outputs, basic triggers.*

## Challenges

| # | Challenge | Concepts |
|---|-----------|----------|
| [1-01](challenges/1-01/) | Morning Briefing | Cron schedules, issue creation, AI summarization |
| [1-02](challenges/1-02/) | Safe & Sound | safe-outputs gate, permission scoping |
| [1-03](challenges/1-03/) | The Watcher | Push event triggers, file-change detection |
| [1-04](challenges/1-04/) | Label Maker | Issue events, AI classification, label API |

## Prerequisites

- Completed [Challenge 00 — Setup](../00-setup/Student/README.md)
- Codespace running with `gh aw` available

## What You'll Learn

- How to write a workflow in Markdown and compile it with `gh aw`
- How scheduled triggers work (cron syntax)
- Why safe-outputs matters and how to enable it
- How to connect an AI engine to a workflow

## Time Budget

~90 minutes. Most squads complete 3 of 4 challenges.
