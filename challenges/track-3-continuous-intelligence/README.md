# Track 3 — Continuous Intelligence 🔴

*Multi-workflow coordination, MCP tools, custom engines, advanced patterns.*

## Challenges

| # | Challenge | Concepts |
|---|-----------|----------|
| [3-01](challenges/3-01/) | The Relay | Workflow chaining, output passing, sequential triggers |
| [3-02](challenges/3-02/) | Context Engine | MCP tools, external data injection, enriched decisions |
| [3-03](challenges/3-03/) | Engine Swap | Multi-engine comparison, prompt portability |
| [3-04](challenges/3-04/) | The Overseer | Meta-workflows, health monitoring, alerting |
| [3-05](challenges/3-05/) | Ship It | End-to-end pipeline: issue → triage → fix → review → merge |

## Prerequisites

- Completed at least 3 challenges from Track 2
- Strong understanding of event triggers, safe-outputs, and AI engine integration

## What You'll Learn

- Chaining workflows via outputs and repository_dispatch
- Using MCP tools to extend workflow capabilities
- Building the same workflow for multiple AI engines
- Creating self-monitoring automation systems
- End-to-end autonomous development pipelines

## Time Budget

~105 minutes. This track is intentionally ambitious. Completing 2-3 challenges is excellent.
