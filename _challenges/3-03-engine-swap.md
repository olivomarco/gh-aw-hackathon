## What you'll build

Three versions of the same workflow (issue categorizer or PR size classifier) — one per engine (Copilot, Claude, Codex). You run them on identical inputs and compare outputs to understand which engine fits which use case and why prompt style must change per engine.

## Concepts

- Engine configuration in gh-aw workflow frontmatter
- Prompt portability and engine-specific writing style
- Reasoning quality, speed, and cost tradeoffs per engine
